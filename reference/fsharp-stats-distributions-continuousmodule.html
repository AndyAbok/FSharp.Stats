studentizedRange(...)
Signature: r:float -> v:float -> c:float -> accuracy:float<MeasureOne> option -> computeParallel:bool -> Distribution<float,float>

Initializes a studentized range distribution.
Accuracy defines the number of steps within the CDF integration (Recommended: 1k-10k, default: 2k). pValue accuracy is minimum 3 digits for v>3. q:qValue r:number of treatments v:df (n-r) c:1.
Integration can be performed in parallel using PSeq
