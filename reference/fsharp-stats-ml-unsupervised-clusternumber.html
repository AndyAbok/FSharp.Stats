silhouetteIndex(clusteredData)
Signature: clusteredData:float [] [] [] -> float

Calculates the silhouette score for a clustered data set where the coordinates of each data point is given as float []. The index ranges from -1 (bad clustering result) to 1 (perfekt clustering result)
silhouetteIndexKMeans(...)
Signature: bootstraps:int -> iClustering:(int -> KClusteringResult<float []>) -> data:float [] [] -> maxK:int -> SilhouetteResult []

The silhouette index can be used to determine the optimal cluster number in k means clustering. bootstraps indicates the number the k means clustering is performed for each k and maxK indicated the maximal cluster number.
